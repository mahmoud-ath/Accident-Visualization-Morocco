
## Introduction
Ce projet est une application de visualisation des accidents de la route au Maroc. Il utilise plusieurs bibliothèques Python pour créer des graphiques interactifs et des cartes afin de représenter les données des accidents.

## Prérequis
Avant de pouvoir exécuter ce programme, assurez-vous d'avoir installé les bibliothèques suivantes :
- pandas
- matplotlib
- geopandas
- tkinter
- tkintermapview
- PySide6
- openpyxl
- shapely

Vous pouvez installer ces bibliothèques en utilisant pip 

##Parametre d'ecran :

Afin de garantir une vision claire et correcte de ce projet assurer d'adjuster les parametre d'ecran suivant :

 1-ouvrire les parametre
 2-acceder au parametre d'ecran :
   - Assurer que la resolution d'ecran est : 1920x1080
   - Assurer que la mise a l'echelle d'ecran est : 100%
